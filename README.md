  	adafruit/Adafruit BMP280 Library@^2.6.8,
	adafruit/Adafruit Unified Sensor@^1.1.14,
	adafruit/DHT sensor library@^1.4.6,
	mobizt/Firebase Arduino Client Library for ESP8266 and ESP32@^4.4.12
